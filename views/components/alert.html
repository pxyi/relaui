弹出层
点击弹出提示框
前端实现：$scope.$emit('prompt', options)
options: object 参数;
1、text: string (必填)
2、href: string（可选）: 点击确定时选择跳转
点击弹出确认框
前端实现：$scope.$emit('confirm', options)
options: object 参数;
1、text: string (可选：默认为“确定删除此数据吗”)
监听确认事件：$scope.$on('confirmTrue', function (e, d){})
点击弹出loading
显示loading: $scope.$emit('loading', true)
隐藏loading: $scope.$emit('loading', false)
